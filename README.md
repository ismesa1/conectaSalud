Despliegue para el Proyecto "ConectaSalud"

1. Microservicios de Autenticación y Reportes:
Están construidos con el framework Laravel. Se necesita un entorno de desarrollo como Laragon y una base de datos MySQL.
Para alzar el proyecto ejecutar con los comandos:
- composer update
- php artisan serv

2. Microservicios de Historias Clínicas y Notificaciones:
Están construidos con el framework Flask. Se necesita tener instalado Python, pip y mongoDB.
Para alzar el proyecto ejecutar con el comando:
- python app.py

3. Microservicio de Gestión de Citas:
Está construido con Java y con Spring Boot. Se necesita tener instalado JDK (Java Development Kit), Apache Maven y MySQL.
Para alzar el proyecto se requiere:
- estar situados en esta dirección c:/conectaSalud/citas/demo.
- ejecutar el comando: ./mvnw spring-boot:run
